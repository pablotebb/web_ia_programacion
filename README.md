# AI & Code Blog

Aplicación web desarrollada con Spring Boot como proyecto de portfolio.

## Tecnologías
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- Bootstrap 5
- H2 Database

## Funcionalidades
- Blog de artículos sobre IA y programación
- CRUD completo de artículos
- Autenticación con Spring Security
- Usuario administrador
- Validación de formularios
